# @seed/models
A Node.js module seed for typescript classes  
## Installation 
```sh
npm install seed-models --save
yarn add seed-models
bower install seed-models --save
```
## Usage
### TypeScript
```typescript
import { Seed } from '@seed/models';
let seed = new Seed();
```

## Test 
```sh
npm run test
```
