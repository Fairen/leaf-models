# @fairen/models
A Node.js module seed for typescript classes  
## Installation 
```sh
npm install @fairen/models --save
yarn add @fairen/models
bower install @fairen/models --save
```
## Usage
### TypeScript
```typescript
import { Seed } from '@fairen/models';
let seed = new Seed();
```

## Test 
```sh
npm run test
```
